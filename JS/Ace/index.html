Ace Editor Module

Ace is a code editor written in JavaScript. This repository has only generated files. If you want to work on ace please go to https://github.com/ajaxorg/ace instead. Here you can find pre-built files for convenience of embedding. It contains 4 versions: - src concatenated but not minified - src-min concatenated and minified with uglify.js - src-noconflict uses ace.require instead of require - src-min-noconflict concatenated, minified with uglify.js, and uses ace.require instead of require For a simple way of embedding ace into webpage see editor.html or list of other simple examples.
