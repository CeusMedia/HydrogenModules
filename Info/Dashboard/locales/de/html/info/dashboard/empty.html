 Was ist ein Dashboard?

Ein Dashboard ist eine Anordnung von mehreren Blöcken mit Informationen in einem Raster.
Diese Blöcke heißen Panels und können jederzeit hinzugefügt, verschoben oder entfernt werden.

Tipp: Man kann mehrere Dashboards erstellen, um zwischen verschiedene Situationen oder Ansichten zu wechseln.


 Was ist ein Panel?

Ein Panel ist ein Block mit Informationen innerhalb eines Dashboards.

Die Inhalte der einzelnen Panels werden aus den vorhanden Informationen zusammen gestellt.
Panels und können innerhalb eines Dashboards jederzeits hinzugefügt, verschoben oder entfernt werden.
