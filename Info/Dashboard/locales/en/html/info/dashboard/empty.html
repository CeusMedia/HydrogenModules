What is a dashboard?

A dashboard is a collection of informational blocks with a grid.
These blocks are called panels and can be added, moved or removed at any time.
You can have several dashboards to switch between views if different situations.

What is a panel?

A panel is an informational block within a dashboard.
The content of these panels are collected from the available informationen within this application.
Panels can be added, moved or removed within a dashboards at any time.
