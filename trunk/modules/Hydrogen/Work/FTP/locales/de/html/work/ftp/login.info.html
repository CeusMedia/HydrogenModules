FTP-Verbindung

Um die Dateien auf dem Server zu sehen, muss eine Verbindung zum FTP-Server hergestellt werden. Bitte gib hier deine Zugangsdaten ein:
Host
IP-Adresse oder DNS-Name des FTP-Servers (ohne ftp://).
Port
Port-Nummer des FTP-Servers, meist 21.
Pfad
Adresse auf dem Server, meist leer.
Benutzer
Name des FTP-Benutzers, bei öffentlichen Servern ist oft auch anonymous möglich.
Passwort
Passwort des FTP-Benutzers, nur leer bei Benutzer anonymous.

Verbindung speichern

Die Verbindung geht mit dem Ausloggen verloren.
Damit die Verbindung automatisch hergestellt wird, musst du deine Zugangsdaten bei deinen Moduleinstellungen speichert.
