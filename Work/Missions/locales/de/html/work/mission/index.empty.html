Aufgaben und Termine

Es wurden keine Einträge gefunden.
Entweder wurden noch keine angelegt oder die Suche mittels der Filter führte ins Leere.

 Aufgaben

Aufgaben beziehen sich auf einen Tag ohne zeitliche Bindung. Entweder sollen sie an diesem Tag begonnen oder beendet werden. Sie werden solange angezeigt, bis sie erledigt wurden.

 Termine

Termine beziehen sich auf einen Tag und eine Uhrzeit.
Aus sie werden solange angezeigt, bis sie erledigt wurden.


Aufgaben können zu Terminen konvertiert werden und umgekehrt.

 Neue Aufgabe  Neuer Termin

Tipp: Synchronisation

Die Aufgaben und Termine können mit Kalender-Programmen, wie z.B. Apple iCal, Microsoft Outlook oder Mozilla Thunderbird, synchronisiert werden.
