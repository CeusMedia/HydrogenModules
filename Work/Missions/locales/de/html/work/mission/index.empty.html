Neue Aufgabe

Aufgaben beziehen sich auf einen Tag. Entweder sollen sie an diesem Tag begonnen oder beendet werden.
Sie werden solange angezeigt, bis sie erledigt wurden.

Neuer Termin

Termine sind wie Aufgaben, aber beziehen sich auf einen bestimmten Zeitraum.
Aufgaben können zu Terminen konvertiert werden und umgekehrt.


Import in Mozilla Thunderbird

Die Aufgaben und Termine können in Kalender-Programmen, wie z.B. Apple iCal, Microsoft Outlook oder Mozilla Thunderbird, importiert werden.
Hier die Anleitung für Mozilla Thunderbird:
  • diesen Link in Zwischenablage kopieren
  • Mozilla Thunderbird öffnen (Erweiterung Lightning muss installiert sein)
  • Menü: Datei -> Neu -> Kalender
  • Quelle "Im Netzwerk" auswählen, weiter
  • Format "iCalendar (ICS)" auswählen, als Adresse den kopierten Link eintragen, weiter
  • Name (z.B. "Aufgaben und Termine") vergeben und Farbe für die Terminblöcke wählen
