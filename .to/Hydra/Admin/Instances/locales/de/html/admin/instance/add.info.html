Informationen

ID und Titel

Die ID ist eindeutiger Schlüssel für eine Instanz und kann nur auch Buchstaben, Zahlen, Unter- und Bindestrich bestehen.
Als Titel kann eine knappe Benennung der Instanz vergeben werden.

Pfad

Der Pfad zur Instanz ist relative zum Wurzelverzeichnis des Webservers und muss existieren.

Konfiguration

Der Pfad zu den Konfigurationsdateien der Instanz ist config/, falls keine andere Angabe gemacht wird.
Als Konfigurationsdatei wird config.ini angenommen, falls nicht anders definiert.
