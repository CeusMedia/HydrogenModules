Für dieses Benutzerprofil kann ein Avatar-Bild verwendet werden. Ein ggfs. automatisch eingesetztes Gravatar-Bild wird dabei ignoriert.

Anforderungen
Maximale Dateigröße: <%maxFileSize%>
Minimale Auflösung: <%minImageSize%>×<%minImageSize%> Pixel
Ideales Bildformat: quadratisch
Das Bild wird automatisch auf eine Quadrat zurecht geschnitten.
