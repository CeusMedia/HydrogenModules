Mit einem sicheren Passwort verhindern Sie, dass Fremde unberechtigt Zugang zu Ihrem Benutzerkonto erhalten.
Sie sollten daher Ihr Benutzerkonto mit einem ausreichend sicheren Passwort schützen.
Falls Ihr Passwort bisher nicht so richtig sicher ist, sollten Sie es jetzt gegen ein sicherers Passwort austauschen.

Tipp: Je sicherer Ihr Passwort ist, umso seltener müssen Sie es ändern.
